# 3-Tier MERN App DevOps Project

This is a fully automated 3-tier application deployment.

## Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Node.js (Express)
- **Database:** MongoDB
- **CI/CD:** Jenkins
- **Quality:** SonarQube
- **Monitoring:** Prometheus & Grafana

## Automatic Builds
This project is configured with GitHub Webhooks to build and deploy automatically on every push.
